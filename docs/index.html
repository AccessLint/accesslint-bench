Features

Lightweight

31 KB gzipped as an IIFE with zero runtime dependencies. Ship accessibility checks without bloating your bundle.

Runs anywhere

Works with happy-dom, jsdom, and real browsers. Run audits in Vitest and React Testing Library using the same environment as your tests.

Main-thread friendly

Time-budgeted processing via createChunkedAudit avoids long tasks on the main thread so your UI stays responsive.

Quick start

npm install @accesslint/core
import { runAudit } from "@accesslint/core";
import { describe, expect, it } from "vitest";

describe("accessibility", () => {
  it("has no violations", () => {
    document.body.innerHTML = `<img src="photo.jpg">`;
    const { violations } = runAudit(document);
    expect(violations).toEqual([]);
  });
});
